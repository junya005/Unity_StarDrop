# StarDrop

星を集めながら飛んでくるUFOをヒップドロップで倒し、スコアを稼ぐカジュアルアクションゲームです。

## 🎮 概要
- **タイトル**: StarDrop
- **開発環境**: Unity, C#
- **開発人数**: 1名（個人制作）
- **制作時期**: 専門学校 2年目
- **ジャンル**: アクションゲーム

## ✨ ゲームの目的・特徴
- **ヒップドロップによる攻撃**: ジャンプからのヒップドロップアクションで、飛来するUFOを撃破します。
- **スターの収集**: ステージ内に配置された星（スター）を集めることでスコアを獲得し、ハイスコアを目指します。
- **爽快感のある演出**: ヒットストップや各種パーティクルエフェクトを活用し、アクション時の爽快感や手応えを高める工夫を行っています。

## 💻 システム・技術的な見どころ
- **マネージャーによるゲーム進行・状態管理**: 
  `InGameManager`、`ScoreManager`、`ResultSceneManager` といったマネージャークラスを分割し、ゲームの進行状況やスコア計算などを適切に管理しています。
- **ヒットストップ機能 (`HitStopManager`)**: 
  敵を倒した瞬間にゲームの進行時間を一瞬止める（ヒットストップ）処理を実装し、攻撃の重みや気持ちよさを表現しています。
- **エフェクト制御 (`EffectController`, `PerticleComponent`)**: 
  アクションに応じた視覚的なフィードバックを強化するため、パーティクルやエフェクトを動的に制御する仕組みを取り入れています。

## 📦 使用ライブラリ・パッケージ
- **UniTask**: 非同期処理の簡略化・最適化
- **DOTween**: UIやオブジェクトのスムーズなアニメーション制御
- **Input System**: プレイヤーの入力制御
- **Universal RP (URP)**: 描画負荷の軽減と高品質なレンダリング
- **TextMeshPro**: 高品質で柔軟なテキスト描画


## 📂 リポジトリ構成 (主要なスクリプト)
`UnityProject/Assets/_Project/Scripts/` 以下に主要なコードを配置しています。
- **Actors/**: プレイヤー、エネミー(UFO)、アイテム(星)などの各アクターの挙動を制御するシステム
- **Managers/**: ゲーム進行、スコア計算、ヒットストップ、シーン遷移などの各種管理クラス
- **UI/**: タイトル画面やリザルト画面のUI制御
